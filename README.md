# 服务器性能检测面板

这是一个用于监控服务器性能的简单面板，使用PHP开发。它可以显示系统信息、CPU占用、内存占用、磁盘空间、网络信息，并允许执行常用命令。

## 使用说明

1. 将这些文件放置在你的服务器上，确保具有PHP运行环境。
2. 打开`index.php`文件，你可以使用任何编辑器进行编辑。
3. 配置面板的样式和布局（如需要），你可以根据需要进行修改。
4. 如果你希望在面板中执行命令，请确保将`execute.php`文件与`index.php`文件放在同一个目录中。
5. 在浏览器中访问`index.php`文件，即可查看服务器性能信息和执行命令。
